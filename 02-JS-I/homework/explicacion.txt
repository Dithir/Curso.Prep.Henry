Las variables son palabras que puedes definir, para que luego puedas asignarles los valores que quieras, ya sean numeros, palabras y es tipo de cosas.
Los strings son un tipo de variable que esta comprendida por palabras, o numeros que se interpretan como palabras y no como numeros.
Las funciones son procesos que puedes definir, para poder hacer un conjunto de muchas cosas con una sola palabra, que sirve mucho en las situaciones en las que tienes que hacerlo varias veces. Asi puedes hacer lo mismo muchas veces, escribiendo la mayor parte una sola vez, y luego una sola palabra por cada vez que quieras hacer lo mismo. Las funciones tienen argumentos que van dentro de parentesis que son datos clave que necesita la funcion para hacer lo el proceso para el que fue hecho, y retorna un valor que se da del resultado de dicho proceso con una palabra redifinida llamada "return".
Las declaraciones if sirven para hacer preguntas logicas, se apoya de simbolos de comparacion como >, < y == (compara si las variables son iguales), que dichas comparaciones devuelven un true o un false, si es true, ejecuta una serie de comandos que el programador puede hacer segun que necesite, como por ejemplo comparar una edad ingresada para ver si es mayor que 18, y devolver una frase que determine si es o no mayor de edad segun el resultado de dicha comparacion.
Los valores booleanos son 2 estados distintos que representan verdadero (true) o falso (false).
